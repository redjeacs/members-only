# Members Only Message Board

## Features:

- Users can register to the the site wtih passwords secured by bcrypt and stored in postgerSQL database
- Logged in users are allowed to add new messages to the board and view all the messages
- Members are given access to message authors and date

## Tools:

- **JavaScript** programming language
- **Node** server-side runtime
- **Expresss** backend framework
- **bcrypt** password hashing
- **PassportJS** user authentication
- **EJS** dynamic views rendering
- **PostgreSQL** database mangement
- **CSS** styling
